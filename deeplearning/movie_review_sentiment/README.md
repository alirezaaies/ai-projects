# 📽️ Movie Review Sentiment Analysis

A **deep learning project** to predict the sentiment of IMDb movie reviews (positive or negative) using **RNN, LSTM, CNN, and Transformer models**. This project is based on **Sebastian Raschka’s “Python Machine Learning” book**, Project 1 – IMDb Sentiment Analysis.  

---

## 📝 Project Overview

The goal of this project is to build models that can classify movie reviews from IMDb as **positive or negative**. The project demonstrates:

- Text preprocessing and tokenization  
- Building and training multiple deep learning models:  
  - Recurrent Neural Network (RNN)  
  - Long Short-Term Memory (LSTM)  
  - Convolutional Neural Network (CNN)  
  - Transformer Encoder  
- Saving the best models based on **validation loss**  
- Logging **training history** (loss and accuracy)  
- Making predictions on new sentences  
- Visualizing **loss and accuracy curves**  

---

## ⚡ Features

- Trainable embedding layer for movie reviews  
- Flexible model selection: RNN, LSTM, CNN, Transformer  
- Automatic saving of **best model** and **training history**  
- Prediction function for **single sentences or batch predictions**  
- Plotting training/validation loss and accuracy
